# SCRIPTIFY 🇮🇳 (SIH 2025 Grand Finalist)

**Problem Statement:** SIH25155 - Transliteration Tool for Street Signs
**Theme:** Heritage & Culture | **Category:** Software
**Team:** NextGen Innovators (Team ID: 100212)

> **Status:** Grand Finalist at Smart India Hackathon 2025.

## 🧐 The Problem
Travelers, pilgrims, and tourists in India often struggle to read local street signs and shop boards due to script barriers. This creates navigation issues and disconnects people from the local culture.

## 💡 The Solution: SCRIPTIFY
Scriptify is an **offline-first AI tool** that breaks language barriers instantly.
**Real-Time Transliteration:** Point the camera at any sign (Hindi, Punjabi, etc.) and read it in your preferred script immediately.
**Offline Capability:** Designed for remote heritage sites where internet access is low.
**Cross-Platform:** Built for Android/iOS using React Native.

# SCRIPTIFY 🇮🇳 (SIH 2025 Grand Finalist)

**Problem Statement:** SIH25155 - Transliteration Tool for Street Signs
**Theme:** Heritage & Culture | **Category:** Software
**Team:** NextGen Innovators (Team ID: 100212)

> **Status:** Grand Finalist at Smart India Hackathon 2025.

## 🧐 The Problem
Travelers, pilgrims, and tourists in India often struggle to read local street signs and shop boards due to script barriers. [cite_start]This creates navigation issues and disconnects people from the local culture[cite: 48, 49].

## 💡 The Solution: SCRIPTIFY
Scriptify is an **offline-first AI tool** that breaks language barriers instantly.
* [cite_start]**Real-Time Transliteration:** Point the camera at any sign (Hindi, Punjabi, etc.) and read it in your preferred script immediately[cite: 19].
* [cite_start]**Offline Capability:** Designed for remote heritage sites where internet access is low[cite: 75].
* **Cross-Platform:** Built for Android/iOS using React Native.

## 🛠️ Tech Stack
* [cite_start]**Framework:** React Native (Expo) [cite: 160]
* [cite_start]**OCR Engine:** Google Vision API / Tesseract (OCR) [cite: 163]
* [cite_start]**Translation Logic:** Custom Character Mapping & Transliteration Algorithms [cite: 203]
* [cite_start]**Backend:** Node.js / Firebase [cite: 162]

## 📸 Project Impact
* [cite_start]**95% Accuracy** in demo conditions[cite: 52].
* [cite_start]**Inclusive Design:** Helps migrants and tourists navigate safely[cite: 110].
* [cite_start]**Heritage Preservation:** Aligns with AICTE-IKS to preserve Indian scripts[cite: 51].
